# Cascade: Selection, Reverse-Engineering and Prediction in Cascade Networks

A modeling tool allowing gene selection, reverse engineering, and
prediction in cascade networks. Jung, N., Bertrand, F., Bahram, S.,
Vallat, L., and Maumy-Bertrand, M. (2014)
[doi:10.1093/bioinformatics/btt705](https://doi.org/10.1093/bioinformatics/btt705)
.
